# Cancer Staging Training Workspace

This directory contains the CT cancer-staging notebook workflow that was previously mixed into `ml/notebooks/`.

## Contents

- `00_environment_check.ipynb` to `05_evaluate_and_gradcam.ipynb`: ordered staging workflow
- `04_train_keras_model.py`: script export of the main training notebook
- `downloads/`: staging-specific dataset download helper notebook
- `experiments/`: staging-related baseline experiments
- `utils/`: helper modules imported by the workflow

## Local Workspace Outputs

These directories are generated by notebook runs and are intentionally git-ignored:

- `processed_data/`
- `models/`
- `figures/`
- `metrics/`

They were kept beside the notebooks so the existing relative-path workflow remains runnable after the move.
